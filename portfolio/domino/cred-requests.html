Согласование кредитных заявок

Назначение

Проведение многоступенчатого процесса согласования кредитных заявок.

Основные возможности

  • Внесение необходимых данных по заявке в зависимости от основных параметров
  • Проведение процесса обсуждения параметров заявки
  • Ведение истории изменения каждого параметра заявки с возможностью быстрого просмотра
  • Подтверждение финальных параметров заявки
  • Основная работа через клиентское приложение IBM Notes
  • Подтверждение доступно через web-интерфейс с адаптивной версткой

Также возможно

  • Интеграция с АБС Банка для импорта/экспорта параметров заявки

Обратите внимание!

Работа приложения рассчитана только в рамках развернутой инфраструктуры сервисов для Notes/Domino

Услуга

Разработка под IBM Notes/Domino (Lotus Notes)

  • OpenNTF Extension Library
