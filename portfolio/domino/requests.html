Заявки

Назначение

Организация сбора, проведения процесса согласования и/или исполнения различных заявок внутри организации

Основные возможности

  • Заведение пользователем заявки через интерфейс Notes-клиента
  • Формирование шаблонов для визирования и согласования заявок
  • Проведение процесса согласования и/или исполнения заявок по шаблонам
  • Информирование участников процесса о необходимости выполнить действие (согласование или исполнение) и стадиях прохождения заявки.
  • Для распределенных систем предусмотрены отложенные уведомления (пользователь не получит уведомление раньше, чем прошла репликация на его почтовый сервер)
  • Просмотр заявок, находящихся на личном рассмотрении
  • Автоматическое присвоение уникального номера заявке с широкой возможностью настройки вида номера
  • Возможность добавления дополнительных визирующих или исполнителей на этапе согласования/исполнения заявки
  • Рассмотрение, согласование и исполнение заявки через Web-интерфейс

Также возможно

  • Взаимодействие с другими приложениями Notes-инфраструктуры
  • Интеграция со внешними системами

Обратите внимание!

  1. Работа приложения рассчитана только в рамках развернутой инфраструктуры сервисов для Notes/Domino
  2. Все необходимые заявки и справочники к ним (если таковые требуются) дорабатываются отдельно.

Услуга

Разработка под IBM Notes/Domino (Lotus Notes)

Технологии

  • LotusScript
  • XPages
  • OpenNTF Extension Library
  • Server Side JavaScript
  • Dojo
