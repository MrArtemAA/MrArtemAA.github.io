Runtime Scripter

Назначение

Утилита для выполнение скрипта или @-формулы "налету" в базе над документом или коллекцией, без внесения изменений в дизайн базы.

Назначение и возможности

  • Выполнение скрипта в нужной базе (документами) без изменения дизайна базы - для небольших правок или пересчетов нет необходимости писать агент в базе
  • Для исполнения скрипта, изменяющего документ достаточно авторского доступа к нему
  • Сохранение написанных скриптов, загрузка ранее сохраненных скриптов
  • Выбор типа скрипта между LotusScript и @-Formula
  • Выполнение скрипта в контекста коллекции документов или одного документа

Утилита, настройка и инструкция по использования доступны на GitHub
