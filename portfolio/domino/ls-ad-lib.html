Библиотека: LS_ActiveDirectory_Lib

Назначение

Библиотека для платформы IBM Notes/Domino, облегчающая взаимодействие и интеграцию с MS Active Directory.

LS_ActiveDirectory_Lib - это библиотека с открытым исходным кодом. Она берет на себя рутинные операции работе с MS Active Directory (MS AD) по протоколу LDAP с помощью провайдера ADSI (Active Directory Service Interface), оборачивая громоздкие вызовы в объекты и методы. Для взаимодействия с ADSI используются его COM-объекты, включенные в семейство ОС Windows.

  • Библиотека доступна на GitHub в рамках проекта LotusScript-Open-Libraries
  • Подробное описание, использование и документация размещена на GitHub
