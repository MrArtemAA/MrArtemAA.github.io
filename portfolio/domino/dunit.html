DUnit

Назначение

Фреймворк модульного тестирования для языка LotusScript платформы IBM Notes/Domino.

Основные возможности

  • Инструменты для написания модульных тестов
  • Встроенный набор проверок
  • Инструменты для расширения "языка" проверок
  • Инфраструктура для запуска тестов
  • Статистика и отчет по результатам прохождения тестов
