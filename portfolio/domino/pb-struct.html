Телефонный справочник и Управление структурой

Назначение

Ведение структуры организации и данных по сотрудникам.

Основные возможности

  • Раздельный вид между телефонным справочником, доступным всем пользователям и управлением структурой
  • Заведение организаций, подразделений и сотрудников
  • Изменение структуры подчинения подразделений
  • Ведение дополнительной и системной информации по сотрудникам, фотография сотрудника
  • Перевод, увольнение сотрудников
  • Ведение сотрудниками контактных данных их персональной карточки
  • Система управления доступами к Notes-приложениям
  • Предоставление интерфейса выбора сотрудника для других Notes-приложений

Также возможно

  • Интеграция с Active Directory (заведение УЗ пользователей, изменение данных)
  • Интеграция с CompanyMedia (перенос в CM данных о сотруднике, фотографии, отображение данных и фотографии в iDocs)
  • Интеграция с основной кадровой подсистемой организации (Импорт данных о подразделениях, сотрудниках, должностях в Notes-приложение)

Обратите внимание!

Приложение является часть (одним из сервисов) инфраструктуры сервисов для Notes/Domino
